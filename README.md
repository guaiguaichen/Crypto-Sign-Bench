本项目旨在对RSA、DSA、ECDSA及Ed25519四种主流数字签名算法进行系统化、多维度的性能与安全性综合评估。通过标准化的测试流程，我们精准测量了各算法在不同参数配置（包括密钥长度、椭圆曲线类型、哈希算法、填充方案及消息大小）下的密钥生成时间、签名与验证时间、签名长度、内存及CPU占用，同时评估了并发处理能力、批量处理效率、错误响应机制以及抗量子攻击、侧信道攻击等安全属性。基于Python密码学库（cryptography）搭建实验环境，累计执行数百组测试用例，并利用描述性统计、方差分析、Tukey事后检验、相关性分析等统计模型对近四万行实验数据进行深度挖掘。结果表明：Ed25519在签名速度、吞吐量、资源消耗及签名长度上全面领先，综合效率最高；ECDSA（尤其是SECP256R1曲线）在安全性与性能间取得良好平衡，适用于区块链及移动设备；DSA性能随密钥长度增长显著下降；RSA虽兼容性广但在长密钥下时间开销大、吞吐量低。此外，我们还给出了各算法的最优参数组合及面向金融交易、物联网、高并发系统等典型场景的选型建议。本项目的评估框架与实验数据可为密码技术选型、系统架构设计及抗量子迁移提供科学、量化的参考依据。
